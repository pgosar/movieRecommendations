This is a website built primarily with React and Flask. It allows the user to input various metrics such as release date and prominent actors and generates a list of movies to recommend to them. It also allows users to input movies they have already watched to find recommendations based on similarities in their plot, genre, and other metrics. For each listing, it generates a list of pros and cons created from parsing movie reviews. 

The website also allows users to create a watchlist of movies organized by various metrics.
